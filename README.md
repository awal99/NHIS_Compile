# NHIS_Compile
a mobile app to compile list of health insurance cards


HOW TO RUN CODE IN DEVELOPMENT

1. Clone/download the code from this repository
2. Extract on your desktop or a convinient location on your computer
3. Open the project in your IDE, example Visual Studio Code
4. If you are using Visual Studio Code, below the IDE there is a terminal run this command to build the node_modules "npm install"
5. You can also use your cmd to cd into the project folder you extracted and run the "npm install" command
6. When the build is complete then we can now run "react-native run-android" for android devices and "react-native run-ios" for ios devices
7. For android make sure your phone is connected and via usb and developer mode enabled.

8. HAPPY CODING!!!
